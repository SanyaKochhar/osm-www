Open Service Mesh (OSM) is a lightweight and extensible cloud-native service mesh that takes a simple approach for users to uniformly manage, secure, and get out-of-the box observability features for highly dynamic microservices environments.

By leveraging the collaboration of the CNCF Envoy network proxy project, OSM provides a simple interface for making configuration changes to your environment.

OSM is guided by 4 principles:

  • Compatible with CNCF Service Mesh Interface (SMI)​
  • Effortless to install, maintain, and operate
  • Simple to understand and contribute to​
  • Flexible enough to handle both simple and complex scenarios through SMI and Envoy XDS APIs
